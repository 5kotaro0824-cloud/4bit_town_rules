## 5. セットアップ

1. 都市ボードをテーブル中央に置く。

2. 建物カード18枚をよくシャッフルし、山札として都市ボードの近くに置く。

3. 山札の上から4枚を引き、都市ボード横に表向きで並べ、  
   **「建設可能列」** を作る。

4. 各プレイヤーは以下を受け取る。

   - 個人ボード 1枚
   - 行動カードのドラフトで獲得する行動カード4枚（後述）
   - ワーカー7個
   - 住居マーカー1個
   - 資源・勝利点用のダイスとキューブ一式

   行動カード16枚は全員共有の山札として使用し、
   このあと行うドラフト手順によって、各プレイヤーが4枚ずつ自分専用の行動カードとして獲得します。
   （2〜3人戦では一部のカードは未使用のまま箱に戻します）。

5. 各プレイヤーのワーカー初期状態：

   - **雇用済みワーカー** …… 3個  
   - **未雇用ワーカー** …… 4個（個人ボードの「未雇用エリア」に置く）

6. 各プレイヤーの木材・石材・コイン・勝利点をすべて 0 にセットする。  
   なお、木材・石材・コインは、以後常に **2個単位** で増減する。

7. **雇用上限の初期値**

   - 全プレイヤーの雇用上限は **3** からスタートする。  
   - 各プレイヤーの住居マーカーを、自分の個人ボード上の「3」のマスに置く。  
     住居マーカーの位置の数値が、そのプレイヤーの雇用上限になる（最大7）。

   ※ 初期状態では「雇用済みワーカー3体」「雇用上限3」でちょうど上限一杯の状態。  
   追加で雇用したい場合、まず家を建てて雇用上限を増やす必要がある。

8. 手番順をランダムに決め、  
   手番順トラックにマーカーを順番に並べる（先頭が第1手番）。

9. 後手補正として初期コインを配る（バランス調整ルール）。

   手番順トラックの先頭から n人目のプレイヤーは、

   > コイン 2×(n-1) 枚

   を受け取る。

   例：

   - 2人プレイ … 1人目 0コイン / 2人目 2コイン  
   - 3人プレイ … 1人目 0 / 2人目 2 / 3人目 4コイン  
   - 4人プレイ … 1人目 0 / 2人目 2 / 3人目 4 / 4人目 6コイン

10. ラウンドマーカーをラウンドトラックの「1」に置く。

11. 行動カードのドラフト

    1. 行動カード16枚をよくシャッフルする。

    2. プレイ人数を n 人とする。16 − 4n 枚を裏向きのままゲームから除外する。
       （2人戦なら8枚除外、3人戦なら4枚除外、4人戦なら除外なし）。

    3. 残りの 4n 枚を各プレイヤーに4枚ずつ配る。

    4. 各プレイヤーは配られた4枚から1枚を選んで手元に確保し、残りを左隣のプレイヤーに渡す。

    5. 同様に、3枚→2枚→1枚と、常に1枚選んで残りを隣に渡すことを繰り返し、
       各プレイヤーが合計4枚を獲得したらドラフト終了とする。

    6. ドラフトで獲得した4枚が、そのプレイヤーの「自分専用の行動カード」となり、
       ゲーム終了まで入れ替わらない。
