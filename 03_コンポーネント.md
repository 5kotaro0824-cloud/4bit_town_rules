## 3. コンポーネント

### 3-1. 都市ボード

都市ボードには、次のエリアが含まれます。

- 場所0〜15のアクションマス **16マス**
- 市役所マス
- 手番順トラック
- ラウンドトラック

### 3-2. 行動カード（全16枚：4bit用）

行動カードは上下に「1側の効果」と「0側の効果」が印刷され、内容の異なる16枚で構成されています。ドラフトで各プレイヤーが4枚ずつ獲得し、その4枚を自分専用の行動カードとしてゲーム終了まで使用します（詳細は 5章）。

4人プレイでは全員のカード構成が異なる非対称なゲームになり、2〜3人戦では一部のカードを未使用とします。

行動カードの表面には、上下に「1側の効果」と「0側の効果」が印刷されています。このゲームでは、自分から見てカードをどちらを「上」にして置くかによって、そのカードの bit 値（0 または 1）が決まります。

- 自分から見て「1側」が上なら、そのカードの bit は **1** です。
- 自分から見て「0側」が上なら、そのカードの bit は **0** です。

4枚それぞれの向きを組み合わせることで、そのラウンドの4bitの状態を表現します（詳細は 4-1, 6-2-2）。

### 3-3. ワーカー

- 各プレイヤーは、自分の色のワーカーコマ（以下「ワーカー」）を **7個** 持ちます。

ワーカーには、次の2種類の状態があります。

- **雇用済み** … このラウンドで使用できるワーカー
- **未雇用** … まだ雇用していないワーカー

維持コストを払わずに「解雇」されたワーカーも、
以降は未雇用ワーカーとして扱います。

### 3-4. 建物カード（全18種）

建物カードは **18種類** あります。
効果の詳細は、8. 建物カード一覧（18種）を参照してください。

### 3-5. 建物カードの配置と上限

建物トークンは使用せず、建物カードそのものを使って
建物の所有を表します。

- 自分の前に表向きで置かれている建物カードが、自分が建設した建物です。

建物カードの予備（計画）や建設の具体的な手順は、
カード③「現場視察／計画」の説明（7章）や、
都市ボードの該当場所効果（9章）を参照してください。

### 3-6. 資源と勝利点

本ゲームで扱う値は次のとおりです。

- 木材
- 石材
- コイン
- 勝利点（VP）

ルール上で **「資源」** と書かれている場合は、

- 木材
- 石材
- コイン

の3種類のみを指し、**勝利点（VP）は含みません。**

### 3-7. 資源の表示方法と上限

各プレイヤーの木材・石材・コイン・勝利点は、

- ダイス：10の位
- ボード上のキューブ：1の位

の組み合わせで表現します。

例：

- ダイス目3 ＋ キューブ2個 → 合計5
- ダイス目0 ＋ キューブ7個 → 合計7

上限は共通で次のとおりです。

> 0 〜 69

70以上になった場合は、即座に **69に切り捨て** ます。

### 3-8. 住居マーカー

- 各プレイヤーは自分の色の住居マーカーを **1個** 持ちます。

住居マーカーは「家の数」と、
**雇用上限（同時に雇用できるワーカー数）** を表すために使用します。

住居マーカーは個人ボード上の「住居トラック」の数値マスに置き、
初期位置は **3**、最大値は **7** です。
家を建てるごとに、住居マーカーを 1 だけ右（大きい数字の方向）に進め、
マーカーが指している数値が、そのプレイヤーの現在の雇用上限になります（詳細は 4-5 を参照）。
